# Peanuts-Discord-Bot
Peanuts bot will send daily Peanuts comics to your server!

## Prefix: `'`

## Commands: 
> `setup` : Setup a channel to send daily comics <br>
> `random` : Get a random Peanuts comic <br>
> `help` : Displays a message with all the commands <br>
